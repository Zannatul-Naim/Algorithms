# Problem: An array is given find the minimum

## Algorithm A

```

1. m = a[1];
2. For i = 2 to size of input:
   a. if m > a[i] then m = a[i];
3. output m;

```

## Algorithm B

comment: Sort the input increasing order and return the first element of the sorted data.

```

1. n = size of the array a
2. sort(a, a+n);
3. output a[1];

```
